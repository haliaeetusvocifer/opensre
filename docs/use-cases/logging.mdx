## Access the logs by going into your run details

OpenSRE automatically captures logs from every tool, task, and environment, so you can debug issues, review performance, or audit results without any manual setup.

## How Logs Work:
Logs are collected through the OpenSRE Agent and sent to OpenSRE's backend log capture service. The OpenSRE Agent reads logs from the local file system and transfers them to our internal services which processes and indexes them to be analyzed later.
