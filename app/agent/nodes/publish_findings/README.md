# Publish Findings Notes For Humans 

Objective:
- The objective for findings publishing is to provide users with the right evidence that they believe that the reporting of the agent is accurate.
- This means we need to collect the right amount of supporting evidence for each claim made. 
- It is very important that we can easily view a split of validated, and unvalidated claims, and develop a (manual) feedback mechanism for us to tweak the system to get into an increasing accuracy of claims overtime.
